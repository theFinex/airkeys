Send into a DAW (Windows)
1. Install loopMIDI (free) → create a port.
2. Here, pick that port in the top-right MIDI out dropdown.
3. In your DAW, set a track's MIDI input to loopMIDI & arm it.
Mac: enable the IAC Driver in Audio MIDI Setup.
